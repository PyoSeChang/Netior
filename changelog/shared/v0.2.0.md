## Features
- `FileEntity`, `FileEntityCreate`, `FileEntityUpdate` 타입 추가
- `CanvasNode`에 `file_id`, `metadata` 필드 추가 (file_path/dir_path 대체)
- `EditorTabType`에 `fileMetadata` 추가
- `EditorTab`에 `canvasId` 필드 추가
- `FILE_*` IPC 채널 6개 추가 (create, get, getByPath, getByProject, update, delete)
- `fileMetadata.*` i18n 키 추가 (ko, en)

## Breaking Changes
- `CanvasNode.file_path`, `CanvasNode.dir_path` 제거 → `file_id`로 대체
- `CanvasNodeCreate.file_path`, `CanvasNodeCreate.dir_path` 제거 → `file_id`로 대체
- `ConceptFile`, `ConceptFileCreate` 타입 제거
- `CONCEPT_FILE_*` IPC 채널 제거
