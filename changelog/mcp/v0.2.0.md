## Features
- `read_pdf_pages`: PDF 페이지 범위의 텍스트 추출 (pdfjs-dist 기반)
- `read_pdf_pages_vision`: PDF 페이지를 이미지로 렌더링 (experimental, canvas 패키지 필요)
- `get_file_metadata`: 파일 엔티티 메타데이터 조회 (기존 TOC 존재 여부 확인용)
- `update_file_pdf_toc`: PDF TOC를 file.metadata.pdf_toc에 저장 (read-merge-write)

## Internal
- `path-validation.ts`로 경로 검증 유틸 추출 (filesystem-tools에서 분리)
- `validateProjectRootPath()` 추가: 프로젝트 root_dir 기준 경로 검증 (module directory 제한 없음)
- pdfjs-dist 의존성 추가

## Cross-Package Impact
- narre-server: `/index` 프롬프트가 read_pdf_pages, get_file_metadata, update_file_pdf_toc 호출
- desktop-app: NarreChat `/index` 워크플로우가 이 도구들을 전제로 동작
- @netior/core: `updateFileMetadataField()` 함수를 update_file_pdf_toc에서 사용
