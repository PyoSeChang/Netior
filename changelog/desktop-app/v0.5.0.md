## Features
- 탭 순회(Ctrl+Tab)와 번호 이동(Ctrl+1~9)이 현재 패인 내에서만 동작 (leaf.tabIds 기준, 드래그 순서 반영)
- 패인 전환 단축키 (Ctrl+Alt+Arrow), main process에서 포워딩하여 터미널에서도 동작
- 활성 패인 시각 표시: 포커스된 패인의 활성 탭에 accent 상단 보더, 멀티패인 시 ring 표시
- 패인 본문 클릭 시 글로벌 activeTabId 동기화 (onMouseDown)
- 탭/패인 전환 시 에디터 자동 포커스 (터미널 xterm, CodeMirror, fallback)
- 에이전트 알림 큐: 미확인 FIFO 큐, Ctrl+.로 가장 오래된 미확인 건 이동, 탭 활성화 시 자동 제거
- 토스트에 미확인 건수 표시 (자기 자신 제외)
