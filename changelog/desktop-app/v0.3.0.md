## Features
- FileMetadataEditor — 파일 고유 속성(description, content_type, topics)과 캔버스 컨텍스트(역할, 관련 페이지)를 한 곳에서 편집
- 노드 컨텍스트 메뉴에 "메타데이터 편집" 추가 (file/dir 노드)
- 캔버스 file/dir 노드가 File 엔티티(file_id FK) 기반으로 동작
- 프로젝트 열 때 File 엔티티 경로 검증 + MissingFilesDialog (삭제/무시 선택)
- fileService (CRUD) + file IPC 핸들러
- FileMetadata 탭에 FileText 아이콘 표시
- Claude 터미널 응답 토스트 알림
- 윈도우 컨트롤 및 파일 트리 UX 개선

## Breaking Changes
- ConceptFile 관련 코드 전면 제거 (IPC, preload, service, UI)
- canvas_nodes의 file_path/dir_path → file_id 기반으로 전환
