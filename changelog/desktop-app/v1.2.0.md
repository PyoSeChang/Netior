## Features
- archetype semantic traits와 system slot 기반 schema 편집을 도입해 temporal 의미를 앱이 직접 이해하고 calendar/timeline에 연결할 수 있게 했습니다.
- day/week/month calendar layout, datepicker navigation, 헤더 통합 컨트롤, all-day와 timed event 분리 렌더링을 추가했습니다.
- recurring event를 virtual occurrence로 투영하고, 편집 시 real concept로 materialize하는 흐름을 추가했습니다.

## Fixes
- Archetype editor에서 legacy session state 때문에 semantic_traits가 배열이 아닐 때 터지던 크래시를 막았습니다.
- Narre transcript/tool rendering의 타입 가드를 정리해 desktop-app 타입체크가 다시 통과하도록 보정했습니다.
- calendar/timeline drag, resize, writeback에서 all_day와 timezone 해석이 어긋나던 경로를 정리했습니다.

## Internal
- legacy field mapping 기반 layout metadata 경로를 제거하고 semantic slot 기반 temporal metadata 조립으로 전환했습니다.
- archetype semantics와 recurring materialization을 위한 migration 020/021을 추가했습니다.
- layout plugin 방향, system semantics trait taxonomy, recurring calendar 설계 문서를 보강했습니다.
