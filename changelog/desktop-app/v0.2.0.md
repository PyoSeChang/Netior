## Features
- 터미널 Todo 오버레이 패널: 터미널별 할 일 관리, 호버 트리거, 핀 고정, 드래그 순서 변경, 계층 구조 지원
- 단축키 레지스트리 및 오버레이: 중앙 단축키 정의, 스코프별 우선순위, Ctrl+/ 오버레이
- 에디터 탭 개별 최소화: 컨텍스트 메뉴에서 단일 탭 최소화 지원 (minimizeSingleTab)
- 앱 테마 변형 추가 (richer theme variants)

## Fixes
- Claude Code 훅 오인식 수정: CLAUDECODE 환경변수 체크 추가로 비-Claude 프로세스 오탐 방지
- EditorDockBar를 position:fixed에서 in-flow 레이아웃으로 변경
- Checkbox 컴포넌트 border 가시성 개선 (border-subtle → border-input)
- DockBar 최소화 탭 간 divider 추가
