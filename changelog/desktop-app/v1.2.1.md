## Features
- concept 노드에 프로필 이미지 기반 비주얼을 추가하고, 탭/사이드바/에디터 헤더까지 같은 시각 표현을 사용하도록 정리했습니다.
- concept 생성/편집 UI에서 아이콘과 프로필 이미지를 라디오로 선택하고, 로컬 파일 선택기로 이미지를 지정할 수 있게 했습니다.
- 노드 제목 길이에 맞춰 기본 카드 폭이 확장되도록 조정하고, 관련 레이아웃의 제목 잘림을 완화했습니다.

## Fixes
- 로컬 파일 기반 프로필 이미지가 `file:///` 로드 제한에 걸리던 문제를 blob URL 방식으로 보완했습니다.
- PDF 워커에서 `URL.parse is not a function` 경고가 발생하던 문제를 워커 전용 폴리필 래퍼로 수정했습니다.
- detached terminal 탭 복귀 시 scroll viewport가 끝까지 복원되지 않던 문제를 repaint 및 viewport sync 보강으로 수정했습니다.
- terminal 링크 hover 시작 위치가 어긋나던 문제를 xterm cell 폭 기준 매핑으로 보정했습니다.

## Internal
- concept 비주얼과 노드 렌더링 유틸을 공통 컴포넌트로 분리해 레이아웃별 중복을 줄였습니다.
- terminal hover/scroll 복원 로직을 xterm wrapper 레이어에서 처리하도록 정리했습니다.
