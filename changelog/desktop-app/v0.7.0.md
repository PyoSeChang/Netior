## Features
- 테마 시스템에 `palette-tint` 도입 — 벽지색(배경 틴트)을 accent에서 분리하여 accent 변경 시 배경 환경이 유지됨
- Button 컴포넌트에 `isActive` prop 추가 — secondary/ghost variant의 활성 상태 지원
- Tailwind config에 `bg-interactive-selected`, `bg-interactive-muted` 토큰 등록

## Fixes
- 깨진 `bg-accent/10` opacity modifier를 정식 semantic token으로 전면 교체 (17개 컴포넌트)
- `reset.css`의 `button { border: none }` → `border-width: 0`으로 변경하여 Tailwind border 클래스 정상 동작
- 설정 모달 좌측 nav 배경을 `surface-sidebar-panel`, 우측 본문을 `surface-panel`로 정렬
- 설정 모달 테마 라이브러리 카드 배경을 `surface-card`로 통일
- 설정 모달 토글 버튼 선택 상태를 `bg-accent text-on-accent`(solid primary)로 강화
- 타이틀바 WindowControls 오른쪽 여백 제거 — 네이티브 Windows 동작과 일치
- detached window 설정 동기화
- float 모드 에디터 레이아웃 폴백 수정
- float 터미널 리사이즈 치수 표시 숨김

## Internal
- 7개 concept CSS에 `--palette-tint-50/100/900/950` 변수 추가
- dark.css/light.css 배경 mixing 변수를 `palette-tint`로 전환
