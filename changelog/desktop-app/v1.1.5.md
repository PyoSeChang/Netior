## Features
- 모듈 생성과 선택 UI를 단일 경로 기준으로 되돌리고, overlay 항목에서 이름 수정과 경로 수정 액션을 직접 제공한다.

## Fixes
- active module에 따라 흔들리던 기본 터미널 `cwd`를 항상 프로젝트 `root_dir`로 고정했다.
- `module.path`와 `module_directories` 호환 레이어를 동기화해서 모듈당 하나의 경로만 유지하도록 정리했다.

## Internal
- 누락된 `019-module-path` 마이그레이션 소스를 복구하고, 모듈 경로 회귀를 검증하는 repository 테스트를 추가했다.
